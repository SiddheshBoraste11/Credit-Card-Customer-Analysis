# 💳 Credit Card Customer Analysis (End-to-End Data Project)

## 📌 Project Overview
This project performs an end-to-end analysis of credit card customers, focusing on **data cleaning, transformation, and visualization** to identify **churn patterns and customer behavior**.

Unlike basic dashboard projects, this solution includes:
- Data cleaning using Excel & SQL
- Database integration using ODBC
- Data modeling & analysis in Power BI

---

## 🔄 Project Workflow

Excel → SQL → Power BI (ODBC Connection) → Dashboard

1. Raw dataset cleaned in Excel  
2. Data imported into SQL database  
3. Data transformed using SQL queries  
4. Connected SQL to Power BI via ODBC  
5. Built interactive dashboard using DAX  

---

## 🛠️ Tools & Technologies
- **Excel** → Data Cleaning & Preprocessing  
- **SQL** → Data Transformation & Querying  
- **ODBC Connector** → SQL to Power BI Connection  
- **Power BI** → Dashboard & Visualization  
- **DAX** → KPI Calculations

## 📊 Key KPIs
- Total Customers: 10,000  
- Active Customers: 8,500  
- Churn Rate: 16.07%  
- Churned Customers: 1,627  
- Avg Transaction: $4.40K  

---

## 🔍 Key Insights

### 📉 Churn Patterns
- Platinum customers have highest churn (~25%)
- High education groups show higher churn
- Inactivity (~2.69 months) is a strong churn signal

---

### 📊 Behavioral Insights
- Blue card users → highest utilization (29%)
- Platinum users → lowest utilization (4.4%)
- Churned customers show reduced activity over time

---

## 💡 Business Recommendations
- Target low-engagement customers
- Personalized campaigns for high-risk segments
- Reward programs for premium customers
- Build churn prediction system

---

## 📌 Conclusion
This project demonstrates an end-to-end data analytics workflow, starting from data cleaning in Excel and SQL, integrating data into Power BI using ODBC, and building interactive dashboards. The analysis identifies key churn patterns, customer behavior trends, and high-risk segments, enabling data-driven strategies to improve customer retention and business performance.
